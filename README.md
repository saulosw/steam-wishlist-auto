## AUTO-STEAM

This project automates the process of collecting and sending information about a user's Steam wishlist.

## Key Features

- Automatic Steam Login
- Wishlist Data Collection
- Email Sending

## Technologies Used

- Python
- Selenium
- BeautifulSoup
- smtplib
- email.mime
- dotenv
